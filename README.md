# АудиоКниги

Библиотека аудио книг.

Библиотека аудио книг. Слушай, скачивай., Проводи время с пользой.

Аудио Книги — это удобное приложение для тех, кто любит слушать книги и не хочет каждый раз искать их по разным сайтам. Здесь всё собрано в одном месте: можно найти книгу, открыть её, начать слушать и потом спокойно продолжить с того места, где остановился.
Приложение сделано с упором на простоту и удобство. Можно искать книги по названию, смотреть авторов, чтецов и жанры, добавлять интересное в Мои книги и быстро возвращаться к тому, что уже слушаешь. Позиция прослушивания сохраняется, поэтому не нужно каждый раз вспоминать, на какой главе остановился.
Если совсем по-простому: это приложение, которое помогает слушать аудио книги спокойно, без лишней суеты и без постоянного хождения по разным сайтам.
Программа ищет аудио книги в интернете постоянно пополняя свою базу, а так же имеет уже встроенную базу ранее найденых книг в количестве 90000 книг
